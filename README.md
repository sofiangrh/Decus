# Decus
A simple metacity theme based on Gentle GNOME's SVG

Includes 8 Metacity themes:
- Transitio
- Transitio Blur
- Lactis
- Lactis Blur
- Pullus
- Pullus Blur
- Lucidus
- Lucidus Blur

# Installation
Put this folder into ~/.themes
